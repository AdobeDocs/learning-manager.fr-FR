# Cours

Lisez cet article pour savoir comment afficher et suivre des cours dans Learning Manager. Participez à des discussions et donnez votre avis.

Les auteurs créent des cours. Les élèves peuvent suivre les cours et les administrateurs peuvent suivre les performances des élèves en fonction de l&#39;utilisation des cours.

## Présentation {#overview}

Adobe Learning Manager permet aux élèves d’accéder aux cours, aux programmes d’apprentissage et aux certifications. Les élèves peuvent parcourir tous les cours disponibles à l&#39;aide du catalogue ou s&#39;inscrire aux cours de leur choix. Les élèves peuvent afficher tous les cours auxquels ils sont inscrits et qui leur sont attribués dans l’onglet Apprentissage.

>[!NOTE]
>
>Les élèves peuvent également installer l’application Learning Manager iPad sur l’Apple Store et l’application Learning Manager Android sur Google Play, et accéder aux cours sur les appareils mobiles. Toutes les fonctionnalités du rôle d’élève sont également disponibles dans l’application. Les élèves peuvent également suivre les cours hors ligne et bénéficier d’un accès transparent après s’être connectés. Reportez-vous à  [Utilisateurs de tablettes iPad et Android](ipad-android-tablet-users.md) pour plus d’informations.

## Affichage des cours {#viewingcourses}

Vous pouvez afficher une liste de tous les cours disponibles en tant qu’élève. Cliquez sur Mon apprentissage dans la page d’accueil ou dans le volet de gauche pour afficher tous les cours auxquels vous êtes inscrit.

![](assets/view-courses.png)
*Afficher les cours attribués*

Si vous n’êtes pas encore en train de démarrer le cours, vous pouvez cliquer sur le bouton Démarrer en regard du cours. Si vous avez déjà commencé à suivre le cours, vous pouvez cliquer sur le bouton Continuer.

Pour afficher un cours terminé, cliquez sur le bouton Revisiter en regard du cours.

Il peut y avoir plusieurs instances d&#39;un cours. Cliquez sur le nom du cours pour afficher les détails. Dans le volet de droite, vous pouvez voir la date d&#39;échéance de chaque instance du cours.

![](assets/learner-course2.png)
*Afficher un cours terminé*

## Efficacité du cours {#courseeffectiveness}

Le score d’efficacité du cours aide les élèves à suivre les cours avec les scores les plus efficaces pour leurs besoins d’apprentissage. L’efficacité du cours est évaluée pour comprendre l’utilité d’un cours pour l’élève. Il est calculé en fonction du retour d’informations reçu par le nombre d’utilisateurs pour un cours particulier. Si le pourcentage d&#39;élèves donnant un retour d&#39;informations est supérieur, l&#39;évaluation de l&#39;efficacité du cours est élevée.

Il s’agit d’une combinaison des résultats des retours d’informations de l’élève sur le contenu du cours, des résultats du quiz de cours pour un élève et du retour d’informations du responsable évaluant un élève en fonction des apprentissages du cours.

Dans la page Cours, un élève peut afficher l&#39;évaluation de l&#39;efficacité des cours sur les vignettes de cours comme indiqué dans l&#39;instantané ci-dessous. L’évaluation de ce cours est de 14.

![](assets/learner-cour-eff-tag1.png)

*Afficher l&#39;évaluation de l&#39;efficacité du cours*

Pour afficher les détails de l&#39;évaluation de l&#39;efficacité du cours, cliquez sur la valeur d&#39;efficacité du cours. Une fenêtre contextuelle s’affiche, comme illustré ci-dessous.

![](assets/learner-course-effectivenss1.png)
*Afficher l&#39;efficacité du cours*

Cliquez sur la flèche vers le bas dans le coin inférieur droit de la fenêtre contextuelle pour afficher comment les calculs d&#39;efficacité de cours sont effectués.

![](assets/course-effectiveness-calculations.png)
*Calcul de l&#39;efficacité du cours*

## Recherche de cours et de programmes d&#39;apprentissage {#searchingcoursesandlearningprograms}

Adobe Learning Manager vous permet de retrouver plus facilement les cours de votre choix rapidement. Vous pouvez rechercher vos cours des manières suivantes :

1. Cliquez sur l’icône de recherche affichée dans le coin supérieur droit. Un champ de recherche s’affiche. Saisissez le nom du cours/programme d&#39;apprentissage ou tous les mots-clés associés à vos cours. Vous pouvez rechercher des cours par métadonnées, notes, compétences, badges ou balises. Les balises sont indexées dans le champ de recherche, ce qui signifie que les balises sont affichées dans le champ de recherche lors de la saisie.
1. L’élève peut affiner les résultats de la recherche dans la page du catalogue en utilisant le type, les compétences, les balises et le statut.

Vous pouvez trier les cours par pertinence, nom ou date de publication en cliquant sur Trier par et en choisissant dans le menu déroulant.

Sur la page des résultats de la recherche, vous pouvez filtrer les cours en fonction de **durée** des cours et de la **format** des cours. Cela vous offre une plus grande flexibilité lors de la recherche de cours et garantit que vous trouvez les cours qui vous conviennent.

## Inscription à des cours {#enrollingforcourses}

Les élèves sont inscrits aux cours de trois manières possibles :

1. L&#39;administrateur/le responsable inscrit certains élèves à des cours obligatoires en fonction des besoins de l&#39;organisation.
1. Le responsable propose certains cours aux membres de son équipe. Vous recevez une notification avec la possibilité d’accepter ou de rejeter le cours/programme d’apprentissage. Lorsque les élèves acceptent une nomination, ils sont inscrits au cours/programme d&#39;apprentissage.
1. Les élèves peuvent directement s&#39;inscrire à un cours/programme d&#39;apprentissage :

   1. Si le cours/programme d’apprentissage est de type auto-inscription, l’élève est inscrit immédiatement.
   1. Si le cours/programme d’apprentissage est de type approuvé par le responsable, l’élève passe à **approbation en attente** état. Après l’approbation du responsable, l’élève est inscrit au cours.
   1. Si les élèves s&#39;inscrivent à un cours figurant sur une liste d&#39;attente (en cas de salle de classe), ils doivent attendre qu&#39;une personne quitte le cours ou qu&#39;un administrateur l&#39;approuve pour le cours.

Les élèves peuvent s&#39;inscrire à une large liste de cours en fonction de leur choix. L’onglet Cours affiche tous les cours auxquels vous êtes inscrit/affectés.

Cependant, vous avez la possibilité de parcourir tous les cours répertoriés dans le catalogue en plaçant le curseur de la souris dessus et en cliquant sur Explorer. La page Inscription s’affiche. Cliquez sur S&#39;inscrire dans le coin supérieur droit de la page pour inclure la liste des cours.

Il peut y avoir plusieurs instances/sessions à un cours/programme d&#39;apprentissage. Dans Catalogue, cliquez sur le nom de la vignette du cours/programme d&#39;apprentissage pour afficher les détails. Vous pouvez voir l&#39;instance d&#39;inscription au cours/programme d&#39;apprentissage en fonction de la date d&#39;échéance de chaque instance du cours/programme d&#39;apprentissage.

**Enregistrer l’intérêt pour les cours**

Vous pouvez vous inscrire pour exprimer votre intérêt pour tous les cours en salle de classe qui n’ont pas de sessions planifiées. Vous recevez une notification chaque fois que la session de cours commence pour rejoindre le cours.

## Workflow en tant qu’élève

En tant qu’élève, vous serez autorisé à évaluer un cours uniquement après l’inscription. Vous pouvez voir la note moyenne pour n’importe quel cours sur la page **Accueil** Page, **Mon apprentissage** page et **Catalogue**.

1. Se connecter en tant que **Élève**. Cette opération charge le fichier **Page d’accueil**.

1. Recherchez un cours en saisissant son nom dans la barre de recherche. Vous pouvez également choisir un cours dans la liste de cours affichée en cliquant sur **Mon apprentissage** ou **Catalogue** dans le volet de gauche.

1. Après avoir sélectionné un cours, cliquez sur **[!UICONTROL S’inscrire]**.

   ![](assets/cp-sr-enroll.png)
   *S&#39;inscrire à un cours*

1. Sélectionnez le nombre d&#39;étoiles pour évaluer un cours en conséquence, 1 étant le moins élevé et 5 le meilleur. Cliquez ensuite sur **[!UICONTROL Envoyer]**.

   ![](assets/cp-sr-srsubmitt.png)
   *Soumettre l’évaluation du cours*

   Un élève peut renvoyer un retour d’informations en soumettant plusieurs fois l’évaluation par étoiles, car seule l’évaluation la plus récente est prise en compte.

1. Un message de confirmation s’affiche après l’envoi de l’évaluation.

   ![](assets/cp-sr-dm.png)
   *Message de confirmation du retour d’informations*

   Si vous souhaitez renvoyer le retour d’informations, vous pouvez le faire en cliquant sur le message. Une fois l&#39;évaluation envoyée, le message s&#39;affiche pendant trois secondes, puis l&#39;évaluation s&#39;affiche. Si vous souhaitez modifier l’évaluation, vous pouvez sélectionner une autre évaluation et la soumettre.

Les cours peuvent être triés en fonction des évaluations fournies. Vous pouvez trier les cours à l’aide de la liste déroulante Trier par située dans le coin supérieur droit, disponible dans la section **Catalogue**.

## Suivre un cours {#consumingnbspacourse}

Après vous être inscrit à un cours, vous pouvez commencer à suivre le cours en cliquant sur Commencer. Vous pouvez également cliquer sur **[!UICONTROL Cours]** dans **[!UICONTROL Formation]** onglet. Choisissez un cours que vous souhaitez commencer à suivre.

Si vous devez encore démarrer un cours, cliquez sur le bouton Démarrer en regard du titre du cours.

Vous pouvez afficher tous les modules du cours dans une fenêtre de navigateur.

**Fonctionnalités du lecteur**

**TDM** - Lors du suivi d&#39;un cours, le lecteur affiche une table des matières pour ce cours particulier dans le volet gauche de la fenêtre. Vous pouvez cliquer sur chaque rubrique et y accéder directement.

**Signets** - Si l&#39;un des modules du cours a une table des matières de module, alors les rubriques dans la table des matières de module peuvent être marquées d&#39;un signet pour référence ultérieure. Lorsqu’elle est marquée d’un signet, une icône en forme de ruban apparaît en regard de l’entrée de table des matières. Les signets peuvent être supprimés en cliquant à nouveau sur le ruban.

**Notes** - Il est possible d&#39;enregistrer des notes pendant que vous suivez le cours. Après avoir enregistré les notes, vous pouvez les enregistrer et les télécharger en tant que PDF ou les envoyer par courrier électronique aux ID de messagerie de tous les élèves inscrits. Lorsque vous cliquez sur Enregistrer, une option vous permet de choisir l’emplacement d’enregistrement en tant que fichier de PDF.

![](assets/notes.png)

*Prendre des notes de cours*

**Sous-titres** - Pour les cours développés dans Adobe Captivate, si les sous-titres sont activés pendant le développement du cours, l’élève peut afficher les légendes. Cliquez sur CC en bas du lecteur. L’option de sous-titrage est disponible uniquement pour le contenu HTML de Captivate 8.0.2. Pour tous les autres types de modules, l’option CC n’apparaît pas sur la barre de lecture.

**Revisiter le cours** - Vous pouvez revoir un cours dans deux modes sous deux scénarios possibles :

* Cliquez sur Revisiter lorsque le cours est à l’état complet.
* Cliquez sur Continuer lorsque le cours est à l’état incomplet.

**Plein écran** Cliquez sur l&#39;icône Plein écran dans le coin inférieur droit du lecteur pour afficher le cours dans la fenêtre en plein écran.

**Boutons de navigation** Vous pouvez cliquer sur les flèches haut/bas pour passer d’une diapositive à l’autre dans le contenu pdf, docx et pptx. Les flèches des diapositives peuvent être utilisées pour passer aux rubriques suivantes pour tous les types de contenu.

**Fermer le cours** Cliquez sur l’icône de fermeture (x) dans le coin supérieur droit du lecteur pour quitter le cours.

*Une fois que vous avez quitté le cours, vous pouvez revenir au cours en cliquant sur le bouton Revisiter sur la page de description du cours.*

## Évaluation par étoiles

Ce n’est qu’après s’être inscrit à un cours qu’un élève peut fournir une évaluation par étoiles pour le cours. Dans la page Présentation du cours, l’élève peut évaluer le cours en attribuant des étoiles (1-note la plus basse, 5-note la plus élevée).

![](assets/star-rating.png)
*Attribution d’une évaluation par étoiles comme commentaire*

Un élève pourra sélectionner une étoile particulière (sur 5) et soumettre l’évaluation. L’élève peut également modifier la sélection en cliquant sur une autre étoile. Une fois l’envoi effectué, un message « Merci pour votre retour d’informations ! » s’affiche.

Si l’élève souhaite renvoyer le retour d’informations, il peut le faire en cliquant sur le message. Le bouton ** Envoyer ** est réactivé. Un élève peut attribuer plusieurs étoiles après s’être inscrit à un cours. La note la plus récente sera toujours prise en compte.

Une fois que l&#39;élève a fourni une évaluation, l&#39;évaluation moyenne par étoiles et le nombre d&#39;élèves qui ont fourni une évaluation apparaissent dans la section **Présentation du cours** page.

![](assets/course-overview.png)
*Présentation du cours*

Cette fonctionnalité est désactivée pour tous les comptes existants. Les administrateurs peuvent l’activer dans Paramètres. Ce n’est qu’ensuite que les étoiles apparaîtront aux élèves.

## Marché de contenus

L’option Marché de contenus est disponible dans le volet gauche de l’application de l’élève. Lorsque vous cliquez sur l’option, vous pouvez voir tous les cours/le catalogue entier et les listes de lecture sélectionnées.

Dans la page ci-dessous, vous pouvez voir les cours présents dans l’ensemble du catalogue. Chaque cours affiche sa durée et le sujet plus large auquel il appartient. Vous pouvez choisir la rubrique dans le filtre situé à gauche de la page.

Vous pouvez prévisualiser un cours pendant exactement deux minutes.

![](assets/catalog.png)
*Aperçu d’un cours sur le marché*

Lorsque l’administrateur vous invite à explorer et prévisualiser un large éventail de cours, une notification s’affiche.

En tant qu’élève, vous pouvez exprimer votre intérêt pour un catalogue entier ou toute liste de lecture sélectionnée dans le **Liste de lecture sélectionnée** section.

![](assets/curated-playlist.png)
*Afficher la sélection*

Une fois que vous avez exprimé votre intérêt pour un cours/une formation, celui-ci est enregistré et l’administrateur peut ensuite le récupérer.

Dans l’application de l’élève, tous les administrateurs ont accès au Marché de contenus. Si l’accès est révoqué par l’administrateur, les élèves ne peuvent pas voir l’onglet Marché de contenus.

Les élèves qui ont été invités par l’administrateur peuvent explorer le Marché de contenus.

>[!NOTE]
>
>Le Marché de contenus n’est pas pris en charge dans Internet Explorer 11.

## Hub de contenu

Le Hub de contenu permet aux administrateurs et aux experts en la matière (SME) de présélectionner les listes de lecture requises dans l’application de l’élève. Une fois la sélection effectuée, les administrateurs peuvent télécharger le formulaire de demande d’achat et le partager avec l’agent commercial Adobe.

Un administrateur peut inviter des experts en la matière à présélectionner la liste de lecture qui les intéresse.

Le Hub de contenu est disponible dans le rôle Élève pour tous les administrateurs. Les administrateurs permettent aux experts en la matière de présélectionner la liste de lecture qu’ils souhaitent acheter.

La page Hub de contenu est toujours visible par les administrateurs dans leur rôle d’élève, car elle leur permet de présélectionner facilement des listes de lecture. Pour vous aider à présélectionner la liste de lecture appropriée, les administrateurs peuvent rendre cette page accessible aux experts en la matière dans leur compte. Visitez la page Formation en entreprise côté administrateur et prenez les mesures nécessaires pour fournir l’accès.

## Choisir une instance de cours {#choosecourseinstance}

Si vous êtes un élève inscrit à une instance de cours dont la session a expiré, vous pouvez désormais basculer vers une autre session pour progresser. Cela n&#39;est possible que si le programme d&#39;apprentissage est flexible.

Pour modifier l&#39;instance de cours, procédez comme suit :

1. Ouvrez Catalogues > Programme d’apprentissage.

   ![](assets/learning-program200s.jpg)
   *Sélectionner un programme d’apprentissage*

1. Utilisez les boutons radio pour choisir si vous souhaitez afficher tous les cours ou les cours avec des instances non sélectionnées. Notez que si l&#39;élève a terminé l&#39;instance, il ne peut pas passer à une autre instance.

   ![](assets/radio-buttons.jpg)
   *Afficher tous les cours ou cours avec des instances non sélectionnées*

1. Le menu déroulant d&#39;instances du cours affiche les instances disponibles. Choisissez l&#39;instance dans la liste déroulante.

   ![](assets/instance-drop-down.png)
   *Sélection d’une instance*

1. Pour appliquer l&#39;instance sélectionnée, cliquez sur **[!UICONTROL Mettre à jour l&#39;inscription]**. L’option Mettre à jour l’inscription est disponible en haut à droite de la page.

   L&#39;instance nouvellement sélectionnée est inscrite. Si un élève est inscrit à une instance de cours qui fait partie d’un autre objet d’apprentissage et s’il met à jour l’instance de cours qui fait partie du programme d’apprentissage flexible, il sera automatiquement désinscrit des autres objets d’apprentissage après un avertissement.

## Terminer un cours {#completingacourse}

En tant qu’élève, vous pouvez terminer le nombre requis de modules d’un cours pour terminer le cours. Les critères d’achèvement du cours dépendent du nombre de modules définis comme obligatoires par l’auteur. Lorsque vous vous inscrivez à un cours avec un minimum de critères d&#39;achèvement, vous pouvez afficher les critères d&#39;achèvement dans le volet droit de la page Cours.

![](assets/learner-xoutofy.png)
*Afficher les critères d’achèvement*

Par exemple, si un cours spécifique comporte les critères d&#39;achèvement d&#39;un module sur deux, le fait de terminer un seul module marque la fin de ce cours. Dans ce cas, lorsque vous terminez le premier module, la barre de progression affiche l’état d’achèvement à 100 %.

Si les modules sont définis dans l’ordre défini par l’auteur, vous devez terminer le nombre requis de modules dans l’ordre séquentiel en commençant par le premier. Si les modules ne sont pas triés, vous pouvez terminer le nombre spécifié de modules dans n’importe quel ordre.

Après avoir terminé un cours avec le nombre requis de modules, si vous souhaitez terminer les modules facultatifs, vous pouvez revoir le cours.

## Affichage et participation aux discussions {#viewingandpariticpatingindiscussions}

En tant qu&#39;élève, vous pouvez interagir avec d&#39;autres élèves et vos instructeurs à l&#39;aide de l&#39;onglet Discussion. Vous pouvez afficher les publications de tout cours affiché ou auquel vous vous inscrivez. Si un administrateur a activé les discussions pour un cours, vous pouvez afficher l&#39;onglet Discussion en regard de l&#39;onglet Notes de ce cours.

Lorsque vous cliquez sur l&#39;onglet Discussions, vous pouvez voir les publications et les commentaires existants pour ce cours. Si vous êtes déjà inscrit au cours, vous pouvez également commencer à saisir des publications ou des commentaires pour que les autres utilisateurs puissent les consulter. Après avoir saisi le message, cliquez sur Publier. Votre post doit contenir au moins 10 caractères.

La publication est immédiatement visible dans l&#39;onglet Discussions. Vous pouvez trier les publications par ordre croissant ou plus anciennes en premier et supprimer celles que vous avez écrites. Même après vous être désinscrit du cours, vous pouvez toujours afficher tous les posts et supprimer les posts que vous avez écrits.

![](assets/learner-discussion.png)
*Afficher les discussions des participants*

L&#39;onglet Discussion n&#39;est pas activé pour les utilisateurs externes.

## Cycle de vie du cours {#courselifecycle}

Un cycle de vie de cours typique se présente comme suit :

**Brouillon** - Lorsqu&#39;un auteur termine la création d&#39;un cours et l&#39;enregistre. À ce stade, le cours n’est pas encore disponible pour les élèves.

**Publié** - Lorsqu&#39;un auteur termine la publication d&#39;un cours. À ce stade, le cours est disponible pour que les élèves s’y inscrivent.

**Retraité** - Après la publication d&#39;un cours, un auteur peut le passer au stade retiré s&#39;il ne veut pas que le cours apparaisse dans le catalogue de cours pour les élèves.

**Supprimé** Adobe - Un cours est à l’état Supprimé lorsqu’il a été complètement supprimé de l’application Learning Manager. Les cours peuvent être supprimés par des auteurs uniquement lorsqu&#39;ils sont à l&#39;état « brouillon » ou « retiré ».

![](assets/lifecycle-03.png)
*Présentation du cycle de vie d&#39;un cours*
