# Application de bureau Adobe Learning Manager

Découvrez comment utiliser l’application de bureau d’Adobe Learning Manager pour créer et enrichir du contenu pouvant être partagé sur l’apprentissage par les réseaux sociaux.

L’application de bureau Adobe Learning Manager est principalement destinée aux élèves qui utilisent l’apprentissage par les réseaux sociaux Learning Manager. Une fois installée sur un ordinateur, l’application permet aux élèves de créer du contenu et de le publier sur des forums dans Learning Manager afin de faciliter un apprentissage informel, collaboratif et social entre pairs. En outre, l’application de bureau garantit également que les élèves reçoivent des notifications d’apprentissage par les réseaux sociaux sur leur bureau, même s’ils ne sont pas nécessairement connectés à l’application Web de Learning Manager.

Les élèves peuvent créer du contenu riche, notamment des vidéos, des fichiers audio et des captures d’écran, et les partager avec d’autres élèves. Le contenu peut être littéralement n&#39;importe quoi - une démonstration ou une présentation du logiciel, une présentation, un podcast, ou une capture d&#39;écran annotée de quelque chose qui peut être partagé.

Sur la page Web Apprentissage par les réseaux sociaux de Learning Manager, cliquez sur Nouvelle publication et sélectionnez enregistrer la vidéo, ou enregistrer l’audio, ou faites une capture d’écran ou accédez à la galerie Learning Manager pour lancer l’application de bureau. Pour plus d’informations sur l’apprentissage par les réseaux sociaux, [cliquez ici](feature-summary/social-learning-web-user.md).

Si les utilisateurs ne disposent pas de l’application de bureau, vous pouvez la télécharger à partir de la page de bureau d’Adobe Learning Manager. Si vous ne parvenez pas à télécharger, [cliquez ici](../kb/troubleshooting-issues-with-adobe-learning-manager-desktop-app.md).

Une fois que l’utilisateur a terminé l’installation de l’application, il doit s’y connecter à l’aide de ses informations d’identification Learning Manager. La page d’accueil s’affiche comme la capture d’écran ci-dessous.

>[!NOTE]
>
>Les élèves ne pourront pas utiliser l’application de bureau si l’apprentissage par les réseaux sociaux est désactivé pour leur compte.

## Création d’une vidéo {#creatingavideo}

L’application de bureau permet à un utilisateur d’enregistrer des données Web et/ou d’écran.

<!--![](assets/video-recordings-08.png)-->
<!--*Record web, screen, or both*-->

Procédez comme suit pour créer un enregistrement vidéo et le publier sur la plateforme web d’apprentissage par les réseaux sociaux :

1. Assurez-vous que la configuration matérielle et logicielle requise est respectée. Voir [Configuration requise](../system-requirements.md) pour information.
1. Pour démarrer un enregistrement à partir de l’Apprentissage par les réseaux sociaux, sélectionnez l’icône d’enregistrement vidéo (web, écran, les deux) qui ouvre l’application de bureau Learning Manager. Dans l’application de bureau avec un utilisateur connecté, le contenu peut être créé, puis publié.
1. Pour commencer l’enregistrement à partir de l’application de bureau, connectez-vous à l’application en utilisant vos informations d’identification Learning Manager en tant qu’élève.
1. La page d’accueil s’affiche.
1. Pour créer une publication, vous pouvez choisir l’une des trois options fournies dans la page d’accueil. Pour ouvrir la fenêtre d’enregistrement, cliquez sur l’icône Enregistrer la vidéo.
1. Selon l&#39;exigence, vous pouvez choisir webcam, écran ou les deux.

   **Webcam** La webcam capture le narrateur qui montre ou présente un projet ou une application, y compris l’audio. Sélectionnez l’icône de webcam parmi les options ci-dessus pour que la fenêtre Live apparaisse.

   **Écran** Enregistrez les actions effectuées sur l’écran du moniteur. En sélectionnant l’icône de l’écran, l’écran est mis en surbrillance avec une boîte de dialogue d’enregistrement au centre de votre écran.

   **Les deux** Enregistre les actions du narrateur et de l’écran. En sélectionnant la dernière icône, l’écran est mis en surbrillance et l’utilisateur reçoit une fenêtre en direct.

1. Définissez les préférences d’enregistrement.

   **Sélectionner la caméra** Cliquez sur l’icône de caméra vidéo en bas de la fenêtre d’enregistrement vidéo. Sélectionnez l’appareil photo que l’utilisateur souhaite utiliser parmi les options proposées.

   **Sélectionner le microphone** Cliquez sur l’icône de microphone au bas de la fenêtre vidéo ou audio et sélectionnez un microphone que l’utilisateur souhaite utiliser parmi les options données.

   >[!NOTE]
   >
   >Si l’utilisateur ne souhaite pas enregistrer de données audio avec un écran ou une vidéo, sélectionnez l’option **Aucun** dans la liste déroulante qui s’affiche en cliquant sur l’icône audio.
   >
   >Lorsque les utilisateurs commencent à enregistrer leur écran ou eux-mêmes, une boîte de dialogue contextuelle s’affiche et leur demande de confirmer s’ils souhaitent continuer à enregistrer sans prise en charge audio. Cliquez sur Oui pour continuer.

1. Utilisez le bouton d’enregistrement pour lancer l’enregistrement. L’enregistrement commence après le compte à rebours (3 secondes).
1. Cliquez sur le bouton Pause ou Reprendre pour suspendre l’enregistrement, puis reprendre si nécessaire. Pour modifier l’enregistrement, voir [Modification d’un enregistrement](adobe-learning-manager-app-for-desktop.md#Editing) et pour publier voir [Publication d’un enregistrement](adobe-learning-manager-app-for-desktop.md#Publishing).

## Création d’un fichier audio {#creatinganaudio}

<!--![](assets/record-audio-2x.png)-->
*Enregistrement audio*

Pour enregistrer un fichier audio, procédez comme suit :

1. Cliquez sur l’icône d’enregistrement audio depuis la fenêtre de lancement de l’application de bureau Learning Manager.
1. Assurez-vous que le volume du microphone est élevé en cliquant sur l’icône de volume du microphone et en ajustant le volume.
1. Cliquez sur le bouton d’enregistrement pour lancer l’enregistrement.
1. Une fois l’enregistrement de l’audio terminé, cliquez sur le bouton Pause pour arrêter l’enregistrement.
1. Pour prévisualiser l’enregistrement, cliquez sur le bouton Lecture .
1. Si l’utilisateur souhaite enregistrer un son supplémentaire dans le même fichier, cliquez sur **[!UICONTROL Reprendre]**. Une fois l’enregistrement souhaité terminé, cliquez sur **[!UICONTROL Suivant]** pour enregistrer le fichier à l’emplacement souhaité. L’enregistrement est enregistré dans l’application Learning Manager d’Adobe pour la galerie de bureau où les utilisateurs peuvent voir leurs enregistrements et leurs captures d’écran.
1. Pour modifier l’enregistrement, voir [Modification d’un enregistrement](adobe-learning-manager-app-for-desktop.md#Editing) et pour publier voir [Publication d’un enregistrement](adobe-learning-manager-app-for-desktop.md#Publishing).

## Création d’une capture d’écran {#creatingascreenshot}

Pour faire une capture d’écran, procédez comme suit :

1. Cliquez sur l’icône de capture d’écran depuis la page d’accueil de l’application de bureau Learning Manager.
1. Pour capturer uniquement une partie de l’écran, cliquez sur l’icône Capturer une partie de l’écran. Pour capturer tout l’écran, cliquez sur l’icône de capture du mode Plein écran.

   <!--![](assets/capture-screenshot.png)
   *Capture a screenshot*-->

1. Pour aider les autres élèves à mieux comprendre et conserver la présentation ou le contenu statique, ajoutez des annotations au contenu.
1. Pour modifier la capture d’écran, [cliquez ici](adobe-learning-manager-app-for-desktop.md#Editing)et pour publier une capture d’écran [cliquez ici](adobe-learning-manager-app-for-desktop.md#Publishing).

## Montage d’une vidéo, d’un fichier audio et d’une capture d’écran {#editing}

Une fois le fichier enregistré, la fenêtre d’édition s’affiche instantanément sur le système où les enregistrements peuvent être rendus parfaits et professionnels à l’aide des options ci-dessous :

## Annotations (enregistrements vidéo et captures d’écran) {#annotationsvideorecordingsandscreenshots}

Les annotations peuvent être utilisées sur des enregistrements vidéo et des captures d’écran pour mettre en évidence des détails spécifiques ou écrire du texte pour que les autres utilisateurs puissent le lire.

>[!NOTE]
>
>Pour les vidéos, les utilisateurs peuvent uniquement annoter en haut de l’écran.

* Les différentes annotations pouvant être sélectionnées sont les suivantes : **ellipse**, **ligne**, **flèche**, **mettre en surbrillance**, et **texte**.

* Choisissez une couleur pour chaque annotation dans le panneau Couleurs du panneau des propriétés de l&#39;annotation.
* Pour **Texte**, sélectionnez les propriétés suivantes : Police, Gras, Italique, Titre, Alignement, Couleur et Couleur d’arrière-plan.

>[!NOTE]
>
>Pour afficher la couleur d’arrière-plan du texte, activez l’option Oui.

## Rogner (tous les types d’enregistrements vidéo et audio) {#trimalltypesofvideoandaudiorecordings}

Pour rogner ou supprimer des parties d’une vidéo ou d’un fichier audio qui ne sont pas nécessaires, utilisez l’outil de rognage.

Pour rogner, cliquez sur **[!UICONTROL Commencer le rognage]** ou cliquez sur l’icône plus sur le montage pour commencer le raccord. Cliquez ensuite sur le bouton Pause du lecteur pour arrêter le rognage. Cliquez sur le bouton **[!UICONTROL Confirmer]** pour terminer le rognage.

Les enregistrements peuvent être rognés en entrant la durée dans le champ **[!UICONTROL Heure de début]** et **[!UICONTROL Heure de fin]** champs de saisie après activation **[!UICONTROL Modifier le minutage]**.

## Panoramique et zoom (tout enregistrement vidéo) {#panampzoomanyvideorecording}

L’action Panoramique et zoom ne peut être effectuée que sur **Les deux** enregistrements vidéo (webcam et écran). Par défaut, l’ensemble de la webcam ou de l’écran est sélectionné. Sélectionnez la poignée d’angle et faites glisser pour sélectionner la partie de l’écran à afficher ou à mettre au point. Double-cliquez sur la fenêtre pour le plein écran.

## Identité visuelle (tous les types d’enregistrements vidéo) {#brandingalltypesofvideorecordings}

Pour créer un nom unique pour une vidéo avec un thème cohérent, sélectionnez l’outil de branding. Le branding vise à établir une présence significative et différenciée dans le système de gestion de l’apprentissage qui attire et retient les élèves pour qu’ils regardent la vidéo complète.

Pour ajouter une marque ou un thème à une vidéo, procédez comme suit :

1. Cliquez sur l’icône de branding dans le panneau de navigation de gauche de la fenêtre d’édition.
1. Sélectionner un **Thème** à partir des options données, les utilisateurs peuvent personnaliser leur propre thème en ajoutant un **début**, **arrière-plan,** et **fin** vidéo de leur système.

   >[!NOTE]
   >
   >Pour télécharger d’autres thèmes, cliquez sur le bouton **Obtenir plus de ressources** bouton.

1. Ajoutez un texte de titre de vidéo et sélectionnez un emplacement sur le **Afficher à** une grille pour qu’elle s’affiche.

   >[!NOTE]
   >
   >Le texte du titre ne peut être ajouté que dans le cadre d’une vidéo de branding. Un thème est requis avant de donner un titre à la vidéo de l’identité visuelle.

1. Ajoutez un nom et un titre de présentateur à la vidéo, si nécessaire. Saisissez un nombre compris dans la plage temporelle de la vidéo dans le champ Afficher à la tête de lecture pour le nom et le titre du présentateur à afficher lors de la lecture de la vidéo.
1. Ajoutez une icône de branding sur le côté gauche ou droit de l’écran vidéo si nécessaire. Pour sélectionner une icône de marque, cliquez sur **[!UICONTROL Sélectionner]** > **[!UICONTROL Parcourir à partir du système]**.
1. Pour prévisualiser la vidéo, cliquez sur le bouton de lecture dans l’écran de prévisualisation.

   ![](assets/branding-options.png)

   *Branding*

## Enregistrement de clips supplémentaires (tous les types d’enregistrements vidéo et audio) {#recordingadditionalclipsalltypesofvideoandaudiorecordings}

Des éléments audio et vidéo supplémentaires peuvent être enregistrés dans un fichier vidéo ou audio existant.

Pour commencer un nouvel enregistrement, cliquez sur l’icône Enregistrer un élément supplémentaire . Le nouvel enregistrement est inséré à l’emplacement de la tête de lecture dans l’enregistrement en cours de modification.

## Publication d’un enregistrement vidéo, d’un enregistrement audio et d’une capture d’écran {#publishing}

Une fois toutes les modifications effectuées, cliquez sur **[!UICONTROL Partager]** > saisissez ou recherchez un **compétence ou forum** qui se rapporte le mieux à la publication > **Post**.

<!--![](assets/share-after-editing-07.png)-->

## Navigation dans la galerie de bureau Learning Manager {#browsingcaptivateprimedesktopgallery}

Tous les enregistrements et les captures d’écran sont enregistrés dans l’application de bureau Adobe Learning Manager. Pour parcourir ces fichiers, cliquez sur **[!UICONTROL Galerie]** dans le panneau de navigation de gauche de la page d’accueil.

>[!NOTE]
>
>La galerie peut également être lancée à partir de l’Apprentissage par les réseaux sociaux en cliquant sur **Galerie Learning Manager** lors de la création d’une publication.

Dans la fenêtre de la galerie, saisissez le nom du fichier que vous recherchez dans le champ de recherche.

Pour créer un enregistrement à partir de la fenêtre de la galerie, cliquez sur le bouton **Enregistrer nouveau** bouton.

Pour modifier ou supprimer un fichier de la galerie, cliquez sur les trois points en bas du fichier et sélectionnez votre option dans la liste contextuelle.

## Notifications {#notifications}

Les notifications dans Learning Manager apparaissent dans la fenêtre de notification, que l’élève soit connecté ou non à l’application web Learning Manager. Les notifications incluent les publications ou les forums que les utilisateurs ont créés, suivent ou auxquels ils ont participé. En cliquant sur la notification, l’utilisateur accède au site Web Apprentissage par les réseaux sociaux de Learning Manager.

Pour désactiver les notifications, cliquez sur **[!UICONTROL Menu Profil*]* > **[!UICONTROL Paramètres] > **[!UICONTROL Notifications silencieuses]**.

## Paramètres dans l’application de bureau Adobe Learning Manager {#settingsinadobecaptivateprimedesktopapplication}

Dans le panneau **Paramètres** , modifiez les paramètres suivants :

Notifications silencieuses, application de lancement automatique, application de mise à jour automatique, activer l’amélioration du produit.

Les utilisateurs peuvent également modifier l’emplacement de la galerie en cliquant sur le bouton **Modifier** lien hypertexte.
