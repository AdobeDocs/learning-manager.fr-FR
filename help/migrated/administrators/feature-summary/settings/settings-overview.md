# Paramètres

## Présentation

Adobe Learning Manager fournit des paramètres d’administration détaillés qui vous permettent de configurer et de personnaliser votre système de gestion de l’apprentissage pour répondre aux besoins spécifiques de votre organisation. Les paramètres sont organisés en plusieurs catégories clés, chacune servant des fonctions administratives distinctes.

### Paramètres de base

* Cette section permet aux administrateurs de configurer les détails organisationnels essentiels, notamment le pays/la région, le fuseau horaire ou les paramètres régionaux.
* Configurez les mécanismes de retour d’informations et les systèmes d’évaluation pour les objets d’apprentissage.
* Divers paramètres généraux qui affectent le comportement et l’utilisation globaux de la plate-forme.

Voir [paramètres](/help/migrated/administrators/feature-summary/settings/basic-settings.md) dans Adobe Learning Manager pour plus d&#39;informations.

### Paramètres d’intégration

* Configurez les options d’authentification, notamment Adobe ID et l’authentification unique (SSO).
* Gérez les intégrations avec les systèmes de données externes pour le provisionnement des utilisateurs, les catalogues de cours et la synchronisation des données de rapport.
* Configurez et gérez les connexions avec d’autres comptes Learning Manager pour le partage de contenu et les initiatives d’apprentissage collaboratif.

Voir Paramètres dans Adobe Learning Manager pour plus d’informations.

### Paramètres avancés

* Créez et gérez des étiquettes personnalisées pour organiser et classer le contenu d’apprentissage, ce qui permet aux élèves de trouver plus facilement des supports pertinents.
* Configurez et gérez les emplacements physiques de la salle de classe pour les sessions de formation dirigée par un instructeur, y compris la capacité, l’équipement et les détails de planification.
* Configurez les tableaux de bord Conformité et Réussite du groupe.

Voir Paramètres dans Adobe Learning Manager pour plus d’informations.
