# Téléchargement, partage et abonnement à un rapport

Après avoir enregistré un rapport dans Report Builder, vous pouvez le télécharger à la demande, le partager avec d’autres administrateurs de votre compte ou vous abonner pour le recevoir à intervalles réguliers.

## Téléchargement d’un rapport

1. Dans l&#39;onglet **Rapports**, recherchez le rapport que vous souhaitez télécharger.
2. Sélectionnez **Télécharger**.
3. Sélectionnez **OK**.
   ![](assets/image046.png)
La génération du rapport s’exécute de manière asynchrone. Vous recevrez une notification dans l’application lorsque le fichier sera prêt.
4. Ouvrez la notification et téléchargez le fichier.

>[!NOTE]
>
>Si votre rapport ne contient pas de données correspondantes, par exemple, parce que les filtres ne renvoient aucun résultat, un fichier vide est toujours généré. Vous ne recevrez pas d’erreur ; le fichier téléchargé aura des en-têtes mais pas de lignes.

## Partage et abonnement à un rapport dans Report Builder

Donnez aux autres administrateurs l’accès à un rapport enregistré et configurez la remise programmée des e-mails dans le Report Builder Adobe Learning Manager.

La boîte de dialogue **Partager et s&#39;abonner** comporte deux sections indépendantes : une pour partager le rapport avec d&#39;autres administrateurs et une pour configurer un abonnement par courrier électronique. Vous pouvez utiliser l’un de ces outils ou les deux.

## Partager un rapport avec d’autres administrateurs

Les administrateurs partagés peuvent afficher, dupliquer et modifier le rapport.

1. Ouvrez le rapport à partager.
2. Sélectionnez **Action** > **Partage et abonnement**.
   ![](assets/image_047.png)
3. Sous **Administrateurs avec accès partagé**, sélectionnez **Modifier**. Sélectionnez ensuite le champ **Sélectionner des utilisateurs/groupes d&#39;utilisateurs**.
4. Recherchez et sélectionnez les administrateurs ou les groupes d’utilisateurs avec lesquels vous souhaitez partager.
   ![](assets/image048.png)
5. Sélectionnez **Enregistrer**.

Les administrateurs sélectionnés peuvent désormais accéder au rapport dans leur vue de Report Builder.

>[!NOTE]
>
>Les administrateurs avec un accès partagé peuvent afficher, dupliquer et modifier le rapport. Pour supprimer l&#39;accès, revenez à **Partager et s&#39;abonner** et désélectionnez l&#39;utilisateur ou le groupe d&#39;utilisateurs.

## Abonnement d’administrateurs à un rapport

Les administrateurs abonnés reçoivent le rapport par e-mail à la fréquence que vous choisissez.

1. Ouvrez le rapport auquel vous souhaitez abonner des administrateurs.
2. Sélectionnez **Actions** > **Partager et s&#39;abonner**.
3. Sous **Administrateurs avec abonnement**, sélectionnez **Modifier**.
4. Sélectionnez la liste déroulante **Fréquence des e-mails**.
5. Choisir une fréquence :
   * **Envoyer quotidiennement**
   * **Envoyer toutes les semaines**
   * **Envoyer tous les mois**
6. Dans le champ **Sélectionner des utilisateurs/groupes d&#39;utilisateurs**, recherchez et sélectionnez les administrateurs ou les groupes d&#39;utilisateurs à abonner.
7. Sélectionnez **Enregistrer**.

Les administrateurs abonnés reçoivent le rapport par e-mail à la fréquence choisie.

>[!TIP]
>
>Appliquez au moins un tri au rapport avant de configurer un abonnement. Cela garantit la cohérence de l&#39;ordre des lignes pour chaque livraison planifiée.
