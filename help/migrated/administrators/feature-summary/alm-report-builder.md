# Report Builder dans Adobe Learning Manager

Générez et téléchargez des rapports personnalisés avec les colonnes, les filtres et les données de votre choix, sans post-traitement à l’aide d’aucun outil.

Le Report Builder de Adobe Learning Manager offre aux administrateurs un canevas de création de rapports en libre-service pour créer exactement les rapports dont ils ont besoin. Au lieu de télécharger un rapport fixe et de le remodeler dans un tableur, vous pouvez sélectionner les colonnes de votre choix, appliquer des filtres et télécharger une sortie propre, le tout depuis un seul endroit.

## Pourquoi le Report Builder a-t-il été introduit ?

Les rapports existants dans Adobe Learning Manager sont corrigés. Chaque rapport comporte une liste définie de colonnes que vous ne pouvez pas modifier et les options de filtrage sont limitées. Les administrateurs qui avaient besoin d&#39;une sortie personnalisée devaient télécharger plusieurs rapports, les joindre dans Excel, appliquer manuellement des filtres et répéter le processus chaque semaine ou chaque mois.

Le Report Builder supprime ce cycle. Vous configurez un rapport une fois, vous l’enregistrez et vous téléchargez de nouvelles données à tout moment, sans avoir à le remodeler.

Le Report Builder résout trois problèmes spécifiques :

* **Colonnes fixes :** les rapports existants incluent de nombreuses colonnes dont vous n&#39;avez peut-être pas besoin et en excluent d&#39;autres. Report Builder vous permet de sélectionner uniquement les champs pertinents pour votre cas d’utilisation.
* **Filtrage limité :** vous pouvez désormais filtrer sur n&#39;importe quel champ pris en charge. Par exemple, inclure la date d’inscription, la date d’achèvement, les champs actifs et les libellés de catalogue, pas seulement le statut.
* **Sources de données incohérentes :** les rapports existants peuvent être extraits de l&#39;interface utilisateur, de l&#39;API de tâche ou des connecteurs, qui peuvent renvoyer des données légèrement différentes en fonction du calendrier. Le Report Builder est extrait d’une source de données unique et cohérente, de sorte que chaque téléchargement reflète les mêmes données sous-jacentes.

## Adaptation du Report Builder aux rapports existants

Report Builder ne remplace pas les rapports fixes existants dans Adobe Learning Manager. Les deux sont disponibles. Utilisez les rapports existants pour les sorties standard sur lesquelles vous comptez déjà. Utilisez le Report Builder lorsque vous avez besoin de colonnes personnalisées, de filtres spécifiques ou de données qui nécessitent actuellement un post-traitement.

## Où trouver du Report Builder

Report Builder disponible sur la page d’accueil de l’administrateur. Sélectionnez **Rapports**, puis **Report Builder**. Deux onglets s&#39;affichent : **Modèles** et **Rapports**.
![](assets/report-builder-001.png)

## Qui peut utiliser Report Builder

Report Builder disponible pour les administrateurs.

>[!NOTE]
>
>Le Report Builder doit être activé pour les comptes. Si vous ne le voyez pas dans la vue Administrateur, contactez l’équipe de votre compte d’Adobe ou CSM.

## Bonnes pratiques

* Commencez avec un modèle si vous débutez avec Report Builder. Les modèles sont préconfigurés pour les cas d’utilisation les plus courants et fonctionnent immédiatement.
* Enregistrez chaque rapport configuré avant de procéder au téléchargement. Les rapports enregistrés peuvent être téléchargés à nouveau ultérieurement sans reconfiguration.
