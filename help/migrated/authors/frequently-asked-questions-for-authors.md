+++Quel est le cycle de vie habituel d’un cours dans Learning Manager ?

Un cycle de vie de cours typique se présente comme suit :

**Brouillon** - Lorsqu&#39;un auteur termine la création d&#39;un cours et l&#39;enregistre. À ce stade, le cours n’est pas encore disponible pour les élèves.

**Publié** - Lorsqu&#39;un auteur termine la publication d&#39;un cours. À ce stade, le cours est disponible pour que les élèves s’y inscrivent.

**Retraité** - Après avoir publié un cours, vous pouvez le passer au stade retiré si vous ne souhaitez pas que le cours apparaisse dans le catalogue de cours pour les élèves.

**Supprimé** Adobe - Un cours est à l’état Supprimé lorsqu’il a été complètement supprimé de l’application Learning Manager. Les auteurs peuvent supprimer le cours uniquement lorsqu&#39;ils sont à l&#39;état « brouillon » ou « retiré ».

+++

+++Comment puis-je publier un projet Captivate dans Learning Manager ?

Vous pouvez publier votre projet Captivate 9 terminé dans Adobe Learning Manager en tant que module. Dans un projet Captivate 9 ouvert, utilisez **Publier** > **Publier sur Adobe Learning Manager** et suivez les instructions pour publier un module.

Reportez-vous à  [Adobe Learning Manager : Publication de modules à partir d’Adobe Captivate 9](http://primehelp.adobe.com/publish-modules-from-adobe-captivate9/)vidéo pour plus d’informations.

Vous pouvez également vous référer à Adobe Captivate 9  [contenu d’aide](http://helpx.adobe.com/captivate/using/publish-project-to-captivate-prime.html) pour la procédure détaillée.

Remarque\
ADOBE CAPTIVATE 9  [contenu d’aide](http://helpx.adobe.com/captivate/using/publish-project-to-captivate-prime.html) est externe à l’application d’aide Learning Manager.

+++

+++Comment choisir les modules de cours dans Learning Manager ?

Cliquez sur  [ici](https://helpx.adobe.com/content/help/en/captivate-prime/authors/how-to-choose-modules.html) pour en savoir plus sur le choix des modules de cours.

+++

+++Quels sont les différents types d’inscription aux cours ?

Lors de la création du cours, vous pouvez choisir le type d’inscription en tant que cours auto-inscrits, nommés par le responsable ou approuvés par le responsable :

**Nomination du responsable** Ces cours ne peuvent être nommés que par les responsables. L’élève ne peut pas s’inscrire à ces types de cours.

**Approbation du responsable requise** Ces cours doivent être approuvés par les responsables. Les élèves peuvent s&#39;inscrire à ces cours, mais ils ne sont pas directement inscrits à ces types de cours sans l&#39;approbation du responsable. Une demande de notification est envoyée aux responsables lorsque les élèves s’inscrivent à ce type de cours. Après approbation du responsable, ces cours seront répertoriés comme étant inscrits pour les élèves.

**Auto-inscription** Les élèves peuvent directement s&#39;inscrire à ces types de cours.

+++

+++ La pagination est-elle prise en charge dans Learning Manager ?

Oui. En mode Auteur, les cours s&#39;affichent au niveau de la page dans les vues Catalogue et Mes cours. En outre, les modules sont affichés au niveau de la page dans la vue Bibliothèque de modules. Au fur et à mesure que vous faites défiler la page vers le bas, d’autres ensembles de cours s’affichent pour la page suivante.

+++

+++Comment puis-je supprimer des cours ?

Les auteurs peuvent supprimer des cours non publiés ou retirés. Après avoir publié un cours, vous pouvez retirer un cours et le repasser à l&#39;état publié. Pour supprimer des cours non publiés, procédez comme suit :

1. Après avoir créé un cours, cliquez sur Mes cours dans le volet de gauche.
1. Passez le curseur de la souris sur le cours que vous souhaitez supprimer et cliquez sur Supprimer le cours.
1. Répondez à la boîte de dialogue de confirmation en cliquant sur OK.

>[!NOTE]
>
>Un cours ne peut pas être supprimé s’il fait partie d’un plan d’apprentissage, d’un programme ou d’une certification ou s’il est ajouté comme condition préalable à un autre cours.

**Cycle de vie du cours**

Un cycle de vie de cours typique apparaît comme suit :

**Brouillon** - Lorsqu&#39;un auteur termine la création d&#39;un cours et l&#39;enregistre. À ce stade, le cours n’est pas encore disponible pour les élèves.

**Publié** - Lorsqu&#39;un auteur termine la publication d&#39;un cours. À ce stade, le cours est disponible pour que les élèves s’y inscrivent.

**Retraité** - Après avoir publié un cours, vous pouvez le passer au stade retiré si vous ne souhaitez pas que le cours apparaisse dans le catalogue de cours pour les élèves.

**Supprimé** Adobe - Un cours est à l’état Supprimé lorsqu’il a été complètement supprimé de l’application Learning Manager. Les cours peuvent être supprimés par des auteurs uniquement lorsqu&#39;ils sont à l&#39;état « brouillon » ou « retiré ».

+++

+++Comment créer des modules pour les cours ?

Créez une bibliothèque de modules qui peuvent être alignés sur les cours en tant que modules d&#39;auto-apprentissage. Les auteurs peuvent créer des modules en suivant les étapes ci-dessous :

1. Cliquez sur Bibliothèque de modules dans le volet de gauche après vous être connecté en tant qu’auteur.
1. Cliquez sur Ajouter dans le coin supérieur droit de la page.
1. Renseignez le nom, la description et les balises du module.
1. Choisir le type de module comme **Partagé(s)** si vous souhaitez partager votre module avec tous les auteurs. Sinon, choisissez privé.
1. Cliquez sur l’icône Télécharger le module et téléchargez le contenu du module.
1. Cliquez sur Enregistrer.

Votre module sera répertorié dans la bibliothèque de modules une fois qu’il aura été chargé avec succès.

+++

+++Que sont les modules de test, de préparation et de contenu ?

L’auteur rencontrerait ces termes lors de la création de cours.

**Modules de test** représentent les modules de base du cours. Si un élève termine ce module important, cela peut être considéré comme l’achèvement du cours même s’il n’a pas terminé le contenu réel du cours.

**Modules de contenu** représente le programme réel du cours. L’élève peut mieux comprendre l’intégralité du contenu du cours en parcourant ces contenus.

**Modules de préparation** aider les élèves à comprendre les bases et à se préparer pour le cours.

+++

+++ Quelle est la différence entre un travail préparatoire et un prérequis ?

L’auteur rencontrerait ces termes lors de la création de cours.

La prérequis correspond à tout cours requis à suivre avant de prendre le cours spécifique. L’auteur peut imposer une condition préalable rendant l’accomplissement obligatoire avant de suivre le cours.

La phase préparatoire correspond à tout module de référence recommandé par l’auteur qui aiderait l’élève à mieux comprendre le cours spécifique. Ce n&#39;est pas obligatoire.

+++

+++ Puis-je créer des programmes d’apprentissage pour mes élèves ?

Les programmes d’apprentissage ne peuvent être créés que par l’administrateur de la société. Contactez l’administrateur de votre organisation.

+++

+++Comment afficher les compétences des élèves et l’état d’achèvement du cours ?

L’auteur ne peut pas afficher les compétences des élèves et l’état d’achèvement du cours. Seul l’administrateur a le droit d’afficher les rapports d’un cours particulier. L’auteur peut contacter l’administrateur pour les rapports.

+++

+++Comment afficher la liste des élèves qui suivent mes cours ?

L’auteur ne peut pas afficher la liste des élèves qui suivent le cours. Seul l’administrateur a les droits d’afficher les élèves inscrits à un cours particulier. L’auteur peut contacter l’administrateur pour la même raison.

+++

+++Puis-je séquencer les modules d’un cours ? Comment ?

L’auteur peut modifier la séquence de modules en faisant glisser la souris sur d’autres modules. Il peut également obliger les élèves à suivre des modules dans l’ordre en sélectionnant **Ordonné** option de séquencement des modules lors de la création du cours.

+++

+++Comment filtrer pour afficher un ensemble spécifique de cours ?

Vous pouvez utiliser l’option Filtrer les cours par état pour filtrer les cours. Pour plus d’informations, voir  [Comment effectuer une recherche dans les cours ?](https://helpx.adobe.com/content/help/en/captivate-prime/authors/frequently-asked-questions-for-authors.html#Course)

+++

+++À quel type de rapports ai-je accès ?

En tant qu’auteur, vous ne pouvez accéder à aucun rapport. Seul l’administrateur a les droits d’afficher/de générer divers rapports. Contactez l’administrateur de votre société pour obtenir des rapports.

+++

+++Comment effectuer une recherche dans les cours ?

Vous pouvez rechercher des cours de deux manières :

1. En utilisant le champ Rechercher affiché dans le coin supérieur droit. Saisissez le nom du cours ou tout mot-clé associé à vos cours pour localiser vos cours.
1. En filtrant la liste des cours à l’aide des filtres. Vous pouvez filtrer les cours par état, par exemple **Tous, Publié, Brouillon** et **Retraité** en cliquant sur chaque option.

   Vous pouvez également effectuer une recherche en fonction des compétences en cliquant sur **Compétences** et choisir chacun d’eux.

   En fonction de votre choix, vous pouvez afficher la liste filtrée des cours et choisir en conséquence.

+++

+++Puis-je ajouter des modules de cours d’autres auteurs à mes cours ? Comment ?

Il existe deux types de modules :

1. Partagé : disponible pour tous les auteurs
1. Privé : disponible uniquement pour l’auteur qui l’a créé.

Ainsi, vous pouvez ajouter uniquement des modules partagés d’autres auteurs à vos cours.

+++

+++Comment ajouter des modules à un cours ?

Vous pouvez ajouter des modules de cours à un nouveau cours ou à un cours provisoire existant. Pour ajouter des modules à un nouveau cours, voir  [Comment créer un cours ?](http://primehelp.adobe.com/questions/how-do-i-create-a-new-course/)

Vous pouvez ajouter des modules à la bibliothèque de modules en cliquant sur **Bibliothèque de modules** et ensuite **Ajouter**.

Pour ajouter des modules à un cours déjà publié, vous pouvez d&#39;abord créer une copie de ce cours, qui est enregistrée en tant que brouillon, puis ajouter des modules.

+++

+++Serai-je averti lorsque les élèves suivront des cours ? Comment ?

Les auteurs ne sont pas avertis lorsque les élèves suivent des cours. L’auteur peut contacter l’administrateur pour obtenir les informations requises.

+++

+++ Puis-je créer des rapports pour les cours ? Comment ?

Dans la version 1.0 d’Adobe Learning Manager, seuls les administrateurs et les responsables peuvent afficher/créer des rapports.

+++

+++Comment créer un quiz pour mon cours ?

Aucune option exclusive n’est disponible pour créer un quiz pour les cours. Vous pouvez le créer dans le cadre du processus de création de contenu.

Le quiz est un contenu SCORM/AICC, Captivate ou Presenter, qui peut être ajouté au cours en tant que module.

+++

+++Comment créer un programme d’apprentissage ?

Les auteurs n’ont pas besoin de créer un programme d’apprentissage pour les cours. Les programmes d’apprentissage sont définis par l’administrateur de votre entreprise.

+++

+++Comment créer un cours ?

Pour créer un cours, procédez comme suit :

1. Connectez-vous à Adobe Learning Manager en tant qu’auteur.
1. Cliquez sur **[!UICONTROL Création de cours]** dans la page de prise en main.\
   La page Informations de base s’affiche.

1. Fournissez les informations de base requises pour le cours. Suivez les instructions de l’aide statique disponible.
1. Cliquez sur **[!UICONTROL Ajouter des modules]** pour suivre un cours et choisir un module dans la liste des quatre modules : fournissez les informations de base requises pour chaque module et cliquez sur **[!UICONTROL Terminé]**.

Une fois la création du cours terminée, cliquez sur Enregistrer. Votre cours est enregistré comme brouillon par défaut. Cliquez sur **[!UICONTROL Publier]** pour publier votre cours.

Pour préparer votre cours à la publication, vous devez ajouter le nom du cours, au moins un module, le type d&#39;inscription, la compétence, le niveau de compétence et les crédits.

Vous pouvez également publier un brouillon de cours répertorié en suivant les étapes ci-dessous :

1. Cliquez sur Mes cours > onglet Brouillons
1. Survolez le cours avec la souris et cliquez sur Publier le cours.

+++

+++Quels sont les formats de cours pris en charge dans Learning Manager ?

Les auteurs peuvent importer les cours e-learning existants créés à l’aide d’Adobe Captivate et d’Adobe Presenter. Les autres formats de cours pris en charge sont les fichiers vidéo, pdf, docx et pptx. Vous pouvez également importer le contenu de cours conforme à AICC.
